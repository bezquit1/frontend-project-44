
# Набор консольных мини-игр. Каждая игра предлагает решить 3 задачи. Если все ответы правильные - вы победили!

# Минимальные требования
1. Node.js версии 14 или выше
2. npm (Node Package Manager) или yarn
3. Bash или Command Line для запуска

# Установка и запуск
1. Клонирование репозитория
# Команда в Bash
   git clone git@github.com:bezquit1/frontend-project-44.git - Клонирование (копирование) репозитория с играми из GitHub
   cd bfrontend-project-44 - переходим в папку с играми
2. Установка зависимостей
# Команда в Bash
npm install - устанавливаются необходимы зависимости между файлами и папками, необходимы для корректной работы игр. 

3. Установка игр в систему
# Команда в Bash
npm link - команда создаст глобальные симлинки, и игры можно будет запускать из любой директории.
# Команда в Bash
## Команды для запуска игр из любой дирректории
1. **Brain Even** - brain-even;
2. **Brain Calc** - brain-calc;
3. **Brain Gcd** - brain-gcd;
4. **Brain Progression** - brain-progression;
5. **Brain Prime** brain-prime;
